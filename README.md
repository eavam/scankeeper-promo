# ScanKeeper marketing website

The public website for [ScanKeeper](https://scankeeper.nomadixapps.org/), an
offline loyalty-card wallet and barcode organizer for iPhone, iPad, and Android.

## What is included

- A conversion-focused home page using real current app screenshots
- Search landing pages for loyalty-card wallets, image scanning, barcode
  organization, and bulk photo import
- Support and privacy pages
- Canonical URLs, Open Graph metadata, `SoftwareApplication`, FAQ, and
  breadcrumb structured data
- `robots.txt`, XML sitemap, web manifest, favicons, and a social preview image
- Automatic deployment to GitHub Pages from `main`

## Local development

Requirements: Node.js 20+ and Corepack.

```bash
corepack enable
yarn install --immutable
yarn develop
```

Open `http://localhost:8000`.

## Production check

```bash
yarn build
yarn serve
```

The static output is written to `public/`. Never commit that directory.

## Updating app screenshots

The source screenshots are generated by the Flutter app repository, not staged
by hand. In `qr_bar`, run the screenshot workflow and copy the current English
iOS captures into this repository:

```text
mocks/marketing/device_screenshots/ios/en-US/01_home.png
  -> src/images/app-home.png
mocks/marketing/device_screenshots/ios/en-US/03_bulk_import.png
  -> src/images/app-bulk-import.png
mocks/marketing/device_screenshots/ios/en-US/08_home_dark.png
  -> src/images/app-home-dark.png
ios/Runner/Assets.xcassets/AppIcon.appiconset/Icon-App-1024x1024@1x.png
  -> src/images/app-icon.png
```

Gatsby generates responsive PNG and WebP variants at build time.
